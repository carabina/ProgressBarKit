  • setProgressBarValue(to:)

    Animates the progress bar from 0 until the given percentage value (in decimal number) of the total width of the progress bar container view.

    This method should only be called after calling setupProgressBar(in:roundingCorners:cornerRadii:) to ensure the progress bar is already initialized.

    Declaration
